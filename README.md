# Data Analysis Project (Netflix Dataset)

## 📌 Objective
To analyze a Netflix dataset using Python and Pandas and extract meaningful insights.

---

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib

---

## 📊 Features Implemented
- Data loading using Pandas
- Data cleaning (handling missing values & duplicates)
- Filtering movies and TV shows
- Grouping data by categories
- Statistical summary
- Data visualization (Bar chart)

---

## 📈 Output
- Movies vs TV Shows bar chart
- Country-wise content analysis
- Rating distribution analysis
- Summary statistics

---

## 📁 Files in this Project
- analysis.py → Main Python script
- netflix_titles.csv → Dataset file
- graph.png → Generated visualization (if saved)

---

```bash
pip install pandas matplotlib
